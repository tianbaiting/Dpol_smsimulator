\doxysubsection{Detailed Description}
Handles coordinate transformations between global and P\+D\+C-\/local frames. 

This class is initialized with the P\+DC\textquotesingle{}s rotation angle around the Y-\/axis and provides methods to transform a T\+Vector3 to and from the P\+DC\textquotesingle{}s local coordinate system.
