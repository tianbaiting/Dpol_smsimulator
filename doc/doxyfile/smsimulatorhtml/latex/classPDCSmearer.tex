\doxysubsection{Detailed Description}
Applies Gaussian position smearing to P\+DC hits. 

This class uses a \mbox{\hyperlink{classPDCRotation}{P\+D\+C\+Rotation}} object to transform hits to the local frame, applies a 2D Gaussian smearing in the local XY plane, and transforms the result back to the global frame.
