## Interpretation for the Main Talk

A direct y-pol Sn112/Sn124 folded-reco isotope plot is now appropriate in the
main talk, provided the scope is stated clearly:

- `124Sn` remains the primary sensitivity target.
- `112Sn` is the isotope reference and now has the same exported tight-px60
  folded-reco observable.
- Both isotopes retain a visible y-pol gamma ordering after folding.
- The Sn112 high-gamma adjacent interval is much narrower than Sn124:
  `g070 -> g080` requires about `1.04e5` usable events for a statistical-only
  3-sigma separation, compared with about `2.77e3` for Sn124.

## Remaining Caveats

- The event-quality selection is still truth-assisted.
- A reco-only isotope closure test is still required.
- Sn112 beamtime/statistics estimates currently use an atom-density scaling
  placeholder rather than a target-specific breakup cross-section extraction.
- Detector and model systematic variations are still not included in the
  statistical-only separation estimates.
